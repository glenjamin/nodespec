Hook

Standard wrapper for hook functions

Requiring

var Hook= require('nodespec/lib/hook').Hook;

In general a Hook should not be instantiated directly, and instead created via Nodespec.before, Nodespec.after, ExampleGroup.before or ExampleGroup.after.
